# EventManager - Плагин событий с сундуками

## Описание
Плагин для Minecraft (Paper/Spigit 1.21), который автоматически создает случайные события с сундуками, содержащими лут разной редкости.

## Возможности
- ✅ Автоматическое создание событий каждые **1.5 часа**
- ✅ Сундуки появляются на координатах из **JSON конфига**
- ✅ **Красивые официальные объявления** с рамками и цветами
- ✅ **Редкость лута отображается** при появлении и открытии сундука
- ✅ Таймер на открытие сундука (1 минута по умолчанию)
- ✅ **Защита сундука:** нельзя открыть или сломать до окончания таймера
- ✅ **Голограмма исчезает** после открытия сундука
- ✅ 4 типа редкости лута: Обычный, Редкий, Эпический, Легендарный
- ✅ Лут случайно распределяется по слотам сундука
- ✅ **2 минуты** на забирание лута с предупреждениями
- ✅ Визуальные эффекты и звуки
- ✅ Команды для управления событиями

## Как работает
1. Плагин запускается и планирует первое событие через **1.5 часа**
2. В случайной локации из JSON конфига появляется сундук с голограммой-таймером
3. Над сундуком отображается обратный отсчет (1 минута)
4. **Сундук защищен:** игроки не могут его открыть или сломать до окончания таймера
5. Через минуту сундук открывается, **голограмма исчезает**, в нем появляется лут случайной редкости
6. Игроки могут забрать лут в течение **2 минут**
7. Сундук исчезает с предупреждениями (1 минута, 30 сек, 10 сек, 5-1 сек)
8. Планируется следующее событие через **1.5 часа**

## Настройка

При первом запуске плагин создаст 3 конфигурационных файла в папке `plugins/EventManager/`:

1. **locations.json** - координаты появления сундуков
2. **messages.json** - все сообщения плагина
3. **loot.json** - настройка лута по редкостям

### 1. Конфигурация координат (locations.json)

```json
{
  "locations": [
    {
      "world": "world",
      "x": 100.0,
      "y": 64.0,
      "z": 100.0
    },
    {
      "world": "world",
      "x": -50.0,
      "y": 70.0,
      "z": 200.0
    }
  ]
}
```

**Как добавить свои координаты:**
1. Откройте файл `plugins/EventManager/locations.json`
2. Добавьте новые локации в массив `locations`
3. Укажите название мира и координаты X, Y, Z
4. Сохраните файл
5. Выполните `/event reload` или перезапустите сервер

**Пример добавления координат:**
```json
{
  "locations": [
    {
      "world": "world",
      "x": 500.0,
      "y": 65.0,
      "z": -300.0
    },
    {
      "world": "world_nether",
      "x": 0.0,
      "y": 100.0,
      "z": 0.0
    },
    {
      "world": "world_the_end",
      "x": 100.0,
      "y": 50.0,
      "z": 0.0
    }
  ]
}
```

### 2. Конфигурация сообщений (messages.json)

Все сообщения плагина настраиваются в этом файле. Используйте `&` для цветовых кодов.

**Пример:**
```json
{
  "event_start": {
    "border": "&6▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬",
    "title": "&e&l⚡ СОБЫТИЕ: СУНДУК С СОКРОВИЩАМИ ⚡",
    "rarity": "&fРедкость лута: {rarity}&l",
    "coords": "&7Координаты: &bX: {x}, Y: {y}, Z: {z}",
    "timer": "&7Открытие через: &c&l1 минуту",
    "hint": "&eПоспешите! Сундук исчезнет через 3 минуты после открытия!"
  }
}
```

**Доступные плейсхолдеры:**
- `{rarity}` - редкость лута
- `{x}`, `{y}`, `{z}` - координаты
- `{time}` - оставшееся время
- `{seconds}` - секунды обратного отсчета

**Цветовые коды:**
- `&0-9, a-f` - цвета
- `&l` - жирный
- `&n` - подчеркнутый
- `&o` - курсив
- `&r` - сброс форматирования

### 3. Конфигурация лута (loot.json)

Настройка предметов для каждой редкости.

**Пример:**
```json
{
  "common": [
    {
      "material": "IRON_INGOT",
      "min_amount": 1,
      "max_amount": 5
    },
    {
      "material": "COAL",
      "min_amount": 5,
      "max_amount": 15
    }
  ],
  "rare": [
    {
      "material": "GOLD_INGOT",
      "min_amount": 1,
      "max_amount": 5
    }
  ]
}
```

**Как добавить предмет:**
1. Откройте `plugins/EventManager/loot.json`
2. Найдите нужную редкость (`common`, `rare`, `epic`, `legendary`)
3. Добавьте новый объект с материалом и количеством
4. Сохраните и выполните `/event reload`

**Список материалов:** https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Material.html

### Настройки времени (LootConfig.java)

### Время событий
```java
public static final int MIN_EVENT_INTERVAL = 5400; // 1.5 часа между событиями
public static final int MAX_EVENT_INTERVAL = 5400; // 1.5 часа между событиями
public static final int CHEST_OPEN_DELAY = 60;     // 1 минута до открытия
public static final int CHEST_LIFETIME = 120;      // 2 минуты жизни сундука
```

Для изменения времени нужно отредактировать `LootConfig.java` и перекомпилировать плагин.

### Настройка лута
Для каждой редкости есть свой метод:
- `getCommonLoot()` - обычный лут (железо, уголь, хлеб)
- `getRareLoot()` - редкий лут (золото, жемчуг края, зелья)
- `getEpicLoot()` - эпический лут (алмазы, алмазная броня)
- `getLegendaryLoot()` - легендарный лут (незерит, звезда ада)

Пример добавления предмета:
```java
loot.add(new LootItem(Material.DIAMOND, 1, 5, LootRarity.EPIC));
//                    материал      мин макс  редкость
```

### Шансы редкости
```java
COMMON(50.0)      // 50% шанс
RARE(30.0)        // 30% шанс
EPIC(15.0)        // 15% шанс
LEGENDARY(5.0)    // 5% шанс
```

## Команды

| Команда | Описание |
|---------|----------|
| `/event start` | Запустить событие немедленно |
| `/event stop` | Остановить все активные события |
| `/event restart` | Перезапустить планировщик событий |
| `/event info` | Показать информацию о настройках |
| `/event reload` | Перезагрузить плагин |

**Права:** `eventmanager.admin` (по умолчанию только для операторов)

## Установка

1. Скомпилируйте плагин:
```bash
./gradlew build
```

2. Скопируйте файл `build/libs/EventManager-0.0.1.jar` в папку `plugins` вашего сервера

3. Перезапустите сервер или выполните `/reload confirm`

4. Настройте координаты, сообщения и лут в папке `plugins/EventManager/`

5. Выполните `/event reload` для применения изменений

## Быстрая настройка

### Изменить координаты событий
1. Откройте файл `plugins/EventManager/locations.json`
2. Измените или добавьте координаты
3. Выполните `/event reload`

### Изменить сообщения
1. Откройте файл `plugins/EventManager/messages.json`
2. Измените нужные сообщения (используйте `&` для цветов)
3. Выполните `/event reload`

### Изменить лут
1. Откройте файл `plugins/EventManager/loot.json`
2. Добавьте/измените предметы в нужной редкости
3. Выполните `/event reload`

## Эффекты

### Визуальные эффекты
- 🎆 Частицы фейерверка при появлении сундука

- 📢 **Красивые рамки** из символов Unicode

- ⏰ **Таймеры и предупреждения** с жирным текстом
- 📍 **Координаты** с подсветкой
